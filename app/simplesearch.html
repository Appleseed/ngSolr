Simple Search

Simple Search is a Javascript tool that provides Apache Solr powered document search for your new or existing web site. The following provides instructions on how to incorporate this functionality.


Add a Search Box to Your Web Pages

Users enter keyword searches on your web site through a search box. The box can appear on one or more pages, and can optionally provide search hints as the user types.

  1. Edit the top of each HTML page that includes a search box so that it looks like this: 
    <!DOCTYPE html>
<html lang="en" ng-app="simplesearch">
    ...
  2. Link the SimpleSearch CSS and Javascript files into your HTML page by including the following in the HEAD: 
    <link rel="stylesheet" type="text/css" href="http://www.esrc.unimelb.edu.au/simplesearch.css" media="screen" />
<script type="text/javascript" src="http://www.esrc.unimelb.edu.au/simplesearch.js"></script>
  3. Add the search box HTML to your page. 
    <!-- simplesearch -->
<searchform data-field="hints"
            data-presenter="search.html"
            data-source="http://www.mysite.esrc.unimelb.edu.au/solr">
    <input name="q" type="text" placeholder="Keyword or placename" searchbox />
    <input name="submit" type="submit" value="search" searchbutton />
</searchform>
<!-- /simplesearch -->
  4. The Network Administrator will provide you with the data-source and data-field values for your web site. These values tell the application which server and index field to fetch the data from. Enter these values into the respective <searchform> attribute fields.

Create a Search Results Page

The search results page executes the search query, then renders the results for the user.

  1. Copy the sample results.html file into your project folder. This file provides you with a minimum layout that you can customize to suit your needs.
  2. The Network Administrator will provide you with the data-source value for your web site. Enter this value into the respective attribute field on the <searchresults> block.
  3. Modify the page style and layout to match your web site presentation. You may optionally modify the presentation of the search results themselves, by editing the contents of the <searchresults> block.

Test Your Search Setup

Ensure that your search boxes and your search results page are now working.

  1. Enter a search term in the search box of any page. After you have typed a minimum of 3 characters, a box should appear below the search term input, with a list of hints. Select a hint and click enter to submit the query.
  2. You should have been directed to the search results page, and zero or more search results should be displayed.
  3. If there is more than one page of results available (more than 10 items by default), then you should see a page list at the bottom of the search results listing. Click each page of the list to confirm that the results update.

If the steps above succeeded, you now have a functioning search service for your web site.
