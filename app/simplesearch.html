Simple Search

Simple Search is a Javascript tool that provides Apache Solr powered keyword search for your new or existing web site. The following provides instructions on how to incorporate this functionality.


Add a Search Box to Your Web Pages

Users enter keyword searches on your web site through a search box. The box can appear on one or more pages, and can optionally provide search hints as the user types.

  1. Edit the top of each HTML page that includes a search box so that it looks like this: 
    <!DOCTYPE html>
<html lang="en" ng-app="simplesearch">
    ...
  2. Link the SimpleSearch CSS and Javascript files into your HTML page by including the following in the HEAD: 
    <link rel="stylesheet" type="text/css" href="http://www.esrc.unimelb.edu.au/simplesearch.css" media="screen" />
<script type="text/javascript" src="http://www.esrc.unimelb.edu.au/simplesearch.js"></script>
  3. Add the search box HTML to your page. 
    <!-- simplesearch -->
<form id="simplesearch-form" ng-controller="SimpleSearchBoxController" ng-submit="submit()">
    <input id="simplesearch-input"
           searchhints
           autocomplete="off"
           name="q"
           ng-model="userquery"
           placeholder='Enter placename or keywords to search for'
           type='text'
           ui-items="getHints(userquery)"
        />
    <button class="btn">Search</button>
</form>
<!-- /simplesearch -->

Create a Search Results Page

The search results page is a web page that will be used to present the results to the user.

  1. Copy the search.html page into your project folder.
  2. Modify the page style and layout to match your web site presentation.
  3. You may optionally modify the presentation of search results as well by editing the HTML block surrounded by the <!-- search results --> comment tags.

Configure the Application

The last step is to configure the application so that it knows where to find the Apache Solr search index for your project.

  1. Open the simplesearch.js file in a text editor. Navigate to approximately line 46 in the file. You should see two variables listed: SOLR_BASE and SOLR_CORE.
  2. Replace the existing values with those provided to you by the network administrator.

Test Your Search Setup

The last step is to test your setup.

  1. Enter a search term in the search box of any page. You should get a drop down box with a list of search hints. Click enter to submit the search query.
  2. You should have been directed to the search results page, and zero or more search results should be displayed.
  3. If there is more than one page of results available (more than 10 items by default), then you should see a page list at the bottom of the search results listing.

If the steps above succeeded, you now have a functioning search setup.
