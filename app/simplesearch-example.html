Simple Document Search

Simple Document Search is a Javascript tool that you can add to your new or existing web site, to quickly and easily incorporate Apache Solr search. The following provides instructions on how to incorporate this functionality.


Add a Search Box to Your Web Pages

Users enter keyword searches on your web site through a search box. The box can appear on one or more pages, and can optionally provide search hints as the user types.

  1. Make sure that your web page is HTML5 compatible. The top of the HTML should look something like this: 
    <!DOCTYPE html>
<html lang="en">
    ...
  2. Copy the simplesearch.js, JQuery, and JQueryUI files into your project folder. Link these files into your HTML page by including the following in the HEAD: 
    <!-- simple search -->
<script type="text/javascript" src="path/to/jquery.js"></script>
<script type="text/javascript" src="path/to/jquery-ui.js"></script>
<script type="text/javascript" src="path/to/simplesearch.js"></script>
  3. Add the search box HTML to your page. 
    <!-- simplesearch -->
<form id="simplesearch-form" ng-controller="SimpleSearchBoxController" ng-submit="submit()">
    <input id="simplesearch-input"
           searchhints
           autocomplete="off"
           name="q"
           ng-model="userquery"
           placeholder='Enter placename or keywords to search for'
           type='text'
           ui-items="getHints(userquery)"
        />
    <button class="btn">Search</button>
</form>
<!-- /simplesearch -->
  4. Add the search app to the HTML tag at the top of the page. 
    <!DOCTYPE html>
<html lang="en" ng-app="simplesearch">
    ...

Create a Search Results Page

The search results page is a web page that will be used to present the search results to the user.

  1. Copy the search.html page into your project folder.
  2. You may optionally modify the presentation of search results by editing the HTML block surrounded by the <!-- search results --> comment tags.

Configure the Search Application

You should now have a functioning search setup on your web site.
